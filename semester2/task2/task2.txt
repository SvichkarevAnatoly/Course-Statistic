Рассмотрим два вида распределений:
нормальное распределение N(x;y)
и смесь нормальных распределений вида 0.9N(x;0,1) + 0.1N(x;0,3).
Как и раньше рассмотрим три объема выборок из данных распределений: 10, 20, 100. 

Рассчитать для данных выборок основные параметры:
1. Медиану 
2. alpha%-Усеченное среднее для разных значений
alpha = 0.05, 0.1, 0.15, 0.2, 0.25

Для смеси нормальных распределений вида (1-epsilon)N(x;0,1)+epsilonN(x;0,1000)
проверить достижение максимально возможного значения пороговой точки медианы при epsilon = 0.5
и найти значение epsilon,
при котором достигается максимально возможное значение пороговой точки
alpha%-усеченного среднего для разных значений alpha = 0.05, 0.1, 0.15, 0.2, 0.25.

Смоделировать хотя бы 1000-2000 выборок каждого объема для получения более показательных результатов.
